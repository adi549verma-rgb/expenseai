# 💰 ExpenseAI

An AI-powered expense tracking and analysis application that helps users manage expenses, visualize spending patterns, and gain financial insights through interactive dashboards.

---

## 🚀 Features

- 📊 Interactive expense dashboard
- 💵 Income and expense tracking
- 📈 Spending trend analysis
- 🥧 Category-wise expense visualization
- 📅 Monthly and yearly reports
- 🤖 AI-powered expense prediction (if applicable)
- 📥 CSV upload support
- 📤 Export reports
